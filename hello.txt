cara
1. git init (buat mulai)
2. git status (buat liat status)
3. git add hello.txt (buat nambahin file hello ke kotak)
4. git add . (buat nambahin file semua yang ada di folder)
5. git status lagi untu cek apakah sudah ditambahkan
6. git commit -m "add hello.txt" (untuk mengunci dan ketika berhasil ada tulisan add hello.txt)
7. git status lagi apakah sudah didalam
8. git remote add origin https://github.com/rzkyff10/belajar-git.git (untuk akses ke github pake link github)
9. git push --set-upstream origin master (lansung ada di githubnya)
10. git push

kalo mau ubah isi, ubah dulu, nanti tambahin ulang ke git dengan cara diatas.
